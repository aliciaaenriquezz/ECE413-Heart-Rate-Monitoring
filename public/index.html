ECE413 Web Development and IoT: Heart Rate Monitoring Project Description


We were tasked with creating a heart rate monitoring IoT web application. Our heart rate and blood oxygen saturation level readings have been obtained using a MAX30102 Pulse Detection Blood Oxygen module. Using this module along with a Particle Argon IoT development board, a user is able to obtain their health metrics through an account. This account can be updated as well.
